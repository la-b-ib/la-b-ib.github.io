CostNest

  • Tech Stack: HTML5, CSS3, JavaScript (ES6+), Materialize CSS, Material Icons, Chrome Manifest V3, Chrome.storage API
  • Github URL: Project Link

CostNest is an advanced Chrome extension designed to track and analyze online shopping expenses in real time. It features a secure PIN lock system and an auto-lock mechanism to safeguard user data and privacy. The extension offers dynamic data visualization with interactive pie and line charts, providing clear insights into spending trends and monthly budgeting. Integrated filtering, sorting, and in-place editing make managing expenses effortless, while backup/restore and CSV export features ensure data portability. With a modern, responsive Material Design interface, CostNest delivers an intuitive and optimized user experience.
