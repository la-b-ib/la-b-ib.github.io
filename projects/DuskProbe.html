DuskProbe

Tech Stack: aiohttp, Keras, Pandas, TensorFlow, spaCy, web3.py, AES-NI, Kubernetes, SQLmap, WPScan

Github URL: Project Link

**DuskProbe** is a powerful Python-based web vulnerability scanner that transforms cybersecurity into a robust, user-friendly experience. With asynchronous scanning, real-time vulnerability detection, and detailed reporting, it empowers users to secure websites effortlessly. Featuring advanced checks for XSS, SQLi, LFI, Web3 contracts, cryptominers, and HTTP/2 vulnerabilities, DuskProbe provides precise metrics on severity, exploit details, and remediation steps. Customize scans with flexible configurations, Tor integration, and secure plugin extensions, all protected by Fernet encryption. Designed for performance with connection pooling and Bloom filters, it ensures ethical scanning with `robots.txt` compliance. Aligned with OWASP standards and cybersecurity best practices, DuskProbe is ideal for developers, pentesters, and security enthusiasts.
